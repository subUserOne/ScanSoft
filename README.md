<h0>SCAN soft</h0>

<h1>Назначение</h1>
<p>Приложение позволяет использовать мобильный телефон как терминал сбора данных (ТСД).
Доступны два режима работы:
<li>оффлайн режим (локальная база номенклатуры)</li>
<li>онлайн режим (подключение к серверу 1С)</li>
